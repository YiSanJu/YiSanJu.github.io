## 一、基础知识

### 1.1 HTML 结构

HTML 是网页的骨架，常用标签包括：

- `<div>` - 块级容器
- `<span>` - 行内容器
- `<a>` - 链接
- `<img>` - 图片

### 1.2 CSS 样式

CSS 用于美化页面，以下是一个简单示例：

```css
.container {
  max-width: 1200px;
  margin: 0 auto;
  padding: 20px;
  background-color: #f5f5f5;
}

.title {
  font-size: 24px;
  color: #333;
  font-weight: bold;
}
```

### 1.3 JavaScript 基础

JavaScript 让网页具有交互性：

```javascript
// 获取元素
const btn = document.querySelector('.btn');

// 添加点击事件
btn.addEventListener('click', () => {
  console.log('按钮被点击了！');
  alert('Hello, World!');
});

// 异步请求示例
async function fetchData() {
  try {
    const response = await fetch('https://api.example.com/data');
    const data = await response.json();
    return data;
  } catch (error) {
    console.error('请求失败:', error);
  }
}
```
